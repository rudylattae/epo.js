tote
  • Specs
  • GitHub

A thin namespaced wrapper for localStorage

    var prefs = tote('preferences');
    prefs.set('theme', 'something');
    prefs.get('theme');
    // 'something'

tote enables you to interact with localStorage as though it is a bunch of small containers where you stash specific things. Each "namespaced" instance writes to the same underlying data store, but does not interfere with any others.
